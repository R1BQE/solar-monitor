\# Solar Activity Monitor by R1BQE



Простая и удобная утилита для радиолюбителей, позволяющая отслеживать состояние ионосферы и условия прохождения радиоволн в реальном времени.



\## Основные возможности

\* \*\*Индексы солнечной активности:\*\* SFI, Sunspots, A-Index, K-Index и другие важные параметры с кратким пояснением, на что они влияют.

\* \*\*КВ Диапазоны:\*\* Прогноз прохождения на любительских диапазонах отдельно для дневного и ночного времени.

\* \*\*УКВ Условия:\*\* Отслеживание таких феноменов, как Аврора и Спорадическое прохождение (E-skip).

\* \*\*Специально для незрячих радиолюбителей:\*\* Интерфейс построен на стандартных компонентах `wxPython`, что обеспечивает отличную доступность для экранных дикторов (NVDA, Jaws).



\## Технический стек

\* \*\*Язык:\*\* Python 3

\* \*\*Библиотека интерфейса:\*\* wxPython

\* \*\*Источник данных:\*\* XML-сервис HamQSL (N0NBH)
